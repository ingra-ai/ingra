# Integrations

Ingra integrates with several third-party services to extend its functionality:

## Google Services
- Access Google Calendar, Contacts, Gmail, Drive, and Sheets through various API scopes.
- Uses OAuth 2.0 for secure access.

## AWS SES
- Utilizes Amazon Simple Email Service for sending emails and notifications.

## Custom APIs
- Integrate custom APIs with Ingra to extend its capabilities using REST or GraphQL.
